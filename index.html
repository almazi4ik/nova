📚 УЧЕБНИК NOVA

🌟 УРОК 1: Переменные

В Nova переменные создаются без ключевых слов:

x = 10
name = "Игрок"
speed = 5

🎨 УРОК 2: Рисование

Создавай круги и квадраты:

create круг 100 100 30 "red"
create квадрат 200 150 40 "blue"

🎮 УРОК 3: Управление

Используй стрелки для движения:

update {
if keyLeft { x = x - 5 }
if keyRight { x = x + 5 }
}

💬 УРОК 4: Вывод текста

Показывай сообщения:

show "Привет, мир!"
show "Счет: " + score

🎯 УРОК 5: Полная игра

playerX = 200
playerY = 150
score = 0
create круг playerX playerY 20 "cyan"

update {
if keyRight { playerX = playerX + 5 }
if keyLeft { playerX = playerX - 5 }

if playerX > 400 { playerX = 400 }
if playerX < 0 { playerX = 0 }

score = score + 1
show "Счет: " + score
}
